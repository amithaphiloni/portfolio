## About

A sleek, dark-themed developer portfolio featuring smooth scroll-triggered animations, a typewriter hero effect, glassmorphism UI cards, and a fully responsive layout. Designed to showcase my projects, skills, and experience.

<br/>

## Features

- **Animated Hero** — typewriter effect cycling through roles, with a glowing animated profile picture
- **About & Skills** — organized skill badges grouped by category with staggered reveal animations
- **Project Showcase** — highlighted project cards with tech stack tags and GitHub links
- **Timeline** — alternating left/right timeline for education and military service
- **Contact Form** — direct-to-WhatsApp contact form with social links
- **Fully Responsive** — mobile-first design that looks great on all screen sizes
- **Smooth Animations** — scroll-triggered Framer Motion transitions throughout

## Getting Started

```bash
# Clone the repository
git clone https://github.com/amithaphiloni/portfolio.git
cd portfolio

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
